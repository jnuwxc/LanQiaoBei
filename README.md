# LanQiaoBei

代码在src目录下，ALGO为算法训练，ADV为算法提高

[蓝桥杯练习网址](http://lx.lanqiao.cn/index.page)

## 题目

题目出自蓝桥杯练习系统中的免费试题

（打算）收录算法训练（ALGO），算法提高(ADV)

（如果有空）也会收录历届试题（PREV）

入门训练和基础训练题量太少且无参考价值，此处不做收录

## 代码

使用java和c++语言描述

基准jdk版本为1.8，虽然蓝桥杯比赛用的好像是1.6

c++为gcc 7.3

（应该）遵循了驼峰命名法和适当的注释

## 解析

每一道题目都有解析

格式为： 

c++代码：题目号.cpp

java代码：题目号.java

解析：题目号.md
